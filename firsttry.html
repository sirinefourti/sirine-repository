    Web Key Components
  1. URL; uniform resource locator

    2. Specifies the exact location of a resource on the Internet. It includes information on how to access that resource. URLs are specific to web resources and are used to identify and locate online content. They are commonly used in web browsers to access websites and web pages.

      Components
    • protocol:Specifies the method or protocol used to access the resource ("http://" or "https://").
    • Domain Name / IP Address: Identifies the location of the server hosting the resource ("www.example.com" or an IP address)
    • Path: Indicates the specific location of the resource on the server ("/page.html").
  2. HTTP :Hypertext Transfer Protocol
    3. class code text comments
    success 200 ok all is well valid recquest
    redirection 301 moved permanently new location automatic
    302 found new location temporarily
    client error 401 unauthorized page recquires login credential
    403 forbidden server will not allow recquest
    404 not found cannot find what it is looking
    server error 500 internal server error otherwise valid recquest

    Dictates the format by which clients request web pages from a server and the format via which servers return information to clients.

  3. Hypertext Markup Language HTML

    4. the markup language used to create and structure web documents. It allows content creators to define the structure and presentation of web pages by using tags and elements. Web browsers interpret HTML to render web content.
